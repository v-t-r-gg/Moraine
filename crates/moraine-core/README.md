# moraine-core

Domain library (no GUI, no HTTP server): Markdown documents, local history, FS watcher, room ids, share URL helpers, run ledger (`*.md.moraine.json`), agent run protocol.

Used by `moraine-cli`, `moraine-mcp`, and `src-tauri`. Product model: [VISION.md](../../VISION.md), [ARCHITECTURE.md](../../ARCHITECTURE.md).
