# Welcome to Moraine

Local-first Markdown editor: Tauri desktop, CLI, history, and optional multiplayer.

## Try

```bash
cargo run -p moraine-cli -- info
cargo run -p moraine-cli -- share examples/welcome.md
npm run dev             # open the printed ?room= URL
npm run tauri:dev
```
